# jarvis-cmd-meteo-weather

Weather command for [Jarvis](https://github.com/alexberardi/jarvis) using [Open-Meteo](https://open-meteo.com).

## Features

- **No API key required** — just install and go
- Current weather conditions (temperature, humidity, wind, feels-like)
- Forecasts up to 16 days out
- City lookup via Open-Meteo geocoding
- IP-based location fallback
- Imperial or metric units

## Setup

1. Install via Jarvis Pantry or manually
2. Optionally configure:
   - **Units**: `imperial` (default) or `metric`
   - **Default Location**: `City,State,Country` (e.g., `Miami,FL,US`)

That's it — no API key, no signup.

## Voice Examples

- "What's the weather?"
- "What's the weather in Chicago?"
- "Will it rain tomorrow?"
- "What's the forecast this weekend?"
